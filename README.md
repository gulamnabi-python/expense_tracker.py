# Smart Expense Tracker (Python)

A simple Python-based expense tracker to manage daily expenses.

## Features
- Add daily expenses
- Categorize expenses
- View all expenses
- Calculate total expense
- Data stored in CSV file

## Technologies Used
- Python
- CSV File Handling

## How to Run
1. Install Python
2. Run:
   python expense_tracker.py

## Sample Output
Add, view and track your expenses easily.
